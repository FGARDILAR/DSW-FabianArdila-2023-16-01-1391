Selectores CSS


Los selectores CSS son herramientas que se utilizan para seleccionar elementos HTML y aplicar estilos específicos a ellos.
Aquí hay algunos ejemplos de selectores CSS comunes: Selector de etiqueta: selecciona todos los elementos con una etiqueta HTML específica. Por ejemplo: "p { color: red; }"
